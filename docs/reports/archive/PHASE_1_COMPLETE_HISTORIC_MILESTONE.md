## **🚀 Quick Start**

```bash
# 1. Install dependencies
cd apps/api && npm install

# 2. Start Redis
brew install redis  # macOS
brew services start redis

# Or Docker
docker run -d -p 6379:6379 redis:latest

# 3. Configure environment
cp .env.example .env
# Fill in:
# - DATABASE_URL
# - JWT_SECRET
# - SENDGRID_API_KEY
# - CLOUDINARY_* (all three)
# - MEILISEARCH_HOST
# - REDIS_HOST
# - SENTRY_DSN (optional)

# 4. Database setup
npm run db:push
npm run db:seed

# 5. Run tests
npm test

# 6. Start development server
npm run dev

# 7. Check health
curl http://localhost:3005/api/health
```

## **📊 System Architecture**

### Backend Stack:
- **Framework**: Hono.js
- **Database**: PostgreSQL + Drizzle ORM
- **Caching**: Redis (IORedis)
- **Search**: MeiliSearch
- **File Storage**: S3/Cloudinary
- **Email**: SendGrid/Nodemailer
- **Monitoring**: Sentry + Winston
- **Testing**: Vitest
- **Security**: CSRF, Rate Limiting, 2FA

### Frontend Stack:
- **Framework**: React 18
- **Router**: TanStack Router
- **State**: Zustand + TanStack Query
- **UI**: Radix UI + Tailwind CSS
- **Build**: Vite with PWA

### Infrastructure:
- **Logging**: Winston (structured)
- **APM**: Sentry (errors + performance)
- **Caching**: Redis (multi-layer)
- **CDN**: Cloudflare/CloudFront
