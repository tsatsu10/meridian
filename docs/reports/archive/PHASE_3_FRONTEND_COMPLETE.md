## 🎯 **User Workflows**

### **1. Create Custom Role:**
```
1. Click "Create Role"
2. Enter name, description, color
3. Select from template (optional)
4. Choose permissions (category-based)
5. Preview role
6. Create & assign users (optional)
```

### **2. Assign Users to Role:**
```
1. Navigate to role details
2. Click "Assign Users"
3. Search and select users
4. Add reason/notes
5. Confirm assignment
6. View in assigned users table
```

### **3. Manage Role Permissions:**
```
1. Open role details
2. View permissions tab
3. See grouped permissions
4. View permission summary
5. Edit (for custom roles)
```

### **4. View Role Activity:**
```
1. Open role details
2. View history tab
3. See audit log
4. Filter by action type
5. View change details
```
