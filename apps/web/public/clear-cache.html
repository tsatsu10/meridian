Clear Service Worker Cache

This tool helps clear cached API URLs in service workers that might be pointing to the old localhost:3005 port.

Manual Steps:

  1. Press F12 to open DevTools
  2. Go to Application tab
  3. Click "Storage" in left sidebar
  4. Click "Clear site data" button
  5. In Service Workers section, click "Unregister" for any workers
  6. Close and reopen the browser
